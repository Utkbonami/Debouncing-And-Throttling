Debouncing is a method we use to limit the number of function invocations we make so that we dont invoke the function everytime we detect a keystroke or change in input, instead we invoke it only after the user has finished giving the input.

Throttling is a method we use to call a function at regular iuintervals while the user is triggering the function invocation, instead of invoking the function for every keystroke we invoke it in regular intervals to detect and account for the changes in the input
